# webprogramming-final

## installation

```
cd client
npm install
(npm install --save apollo-boost react-apollo graphql)
```

```
cd server
npm install
(npm install --save apollo-server graphql)
```

## To Run

* client
  ```
  npm run client
  (yarn client)
  ```
* server
  ```
  npm run server
  (yarn server)
  ```
